
Desenvolver uma interface web que exiba a situação atual do combustível de uma lista de equipamentos, conforme o layout abaixo:

Esse layout é apenas um exemplo, vocês tem liberdade para adaptá-la da forma que acharem mais usável. A tela deve ser responsiva e capaz de lidar com vários equipamentos.

Os dados estão no arquivo equipamentos.json.

A cor do equipamento segue a seguinte regra:
Combustível <= 30%: Vermelho

Combustível >30% e <= 60% Amarelo

Combustível > 60% Verde

Desafio 02 (Diferencial):

Em anexo seguem também uma lista de arquivos (dados_historicos.zip), que contém o histórico de cada um dos equipamentos. Essa parte do desafio é um diferencial e consiste em fazer uma 
nova tela que exibe esses dados em forma de gráfico, e será exibida ao clicar sobre algum dos equipamentos.
